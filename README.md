# Parcel Service System

A comprehensive database design project for a **parcel delivery and tracking system**. This project documents the complete database architecture, schema design, and specifications for a courier application that manages parcel shipments, deliveries, and real-time tracking.

## 📋 Overview

This repository contains the database design specifications for a parcel service platform. The project includes detailed documentation of:

- **Database Schema** - Entity relationships, tables, and data structures
- **System Requirements** - Functional and non-functional requirements
- **Database Design** - Logical and physical database models
- **UI/UX Prototype** - Figma design for the courier tracking application
- **Project Presentation** - Video walkthrough of the entire system

## 🎯 Key Features

The parcel service system covers:

- **Customer Management** - User profiles, account information, contact details
- **Parcel Tracking** - Real-time status updates and location tracking
- **Delivery Management** - Shipment scheduling, route optimization, delivery confirmation
- **Courier Operations** - Courier profiles, assignment, performance metrics
- **Payment & Billing** - Transaction records, invoicing, shipping costs
